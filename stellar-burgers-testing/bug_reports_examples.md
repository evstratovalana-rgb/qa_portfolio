**Модуль:** Аутентификация и лента заказов

## Критические дефекты (Blocker/Critical)

### BLK-70: Некорректный переход при вводе несуществующей почты
**Краткое описание:**  
При вводе email, отсутствующего в БД, система вместо показа ошибки пытается перейти на страницу сброса пароля.

**Шаги воспроизведения:**
1. Перейти на `/forgot-password`
2. В поле "Укажите e-mail" ввести `92elistrtova@gmail.com`
3. Нажать кнопку "Восстановить"

**Ожидаемый результат:**  
Показ ошибки "Пользователь с таким e-mail не найден", остаемся на той же странице.

**Фактический результат:**  
Происходит переход на страницу сброса пароля.

**Окружение:** Яндекс.Браузер 25.4.1.1024 (1280×720), Firefox 137.0.2 (1920×1081)

## Блокирующие дефекты (Blocker)

### BLK-86: Чёрный экран при сбросе пароля с неверным кодом
**Краткое описание:**  
При вводе неверного кода подтверждения и корректного пароля появляется чёрный экран вместо сообщения об ошибке.

**Шаги воспроизведения:**
1. Перейти на `/forgot-password`
2. Ввести email `evstratova1212@mail.ru`
3. Нажать "Восстановить"
4. На странице сброса ввести:
   - Пароль: `1perelom2^`
   - Неверный код подтверждения
5. Нажать "Сохранить"

**Ожидаемый результат:**  
Показ ошибки "Неверный код".

**Фактический результат:**  
Чёрный экран сайта.

**Приоритет:** Блокирующий  
**Окружение:** Яндекс.Браузер 25.4.1.1024 (1280×720), Firefox 137.0.2 (1920×1081)

## Стандартные дефекты (Major/Normal)

### BLK-68: Отсутствие индикатора загрузки при входе
**Краткое описание:**  
При нажатии кнопки "Войти" не отображается троббер (индикатор загрузки).

**Шаги воспроизведения:**
1. Перейти на `/login`
2. Ввести email: `qwgdtf@gmail.com`
3. Ввести пароль: `wedrt34`
4. Нажать "Войти"

**Ожидаемый результат:**  
Появление троббера в виде расходящихся кругов.

**Фактический результат:**  
Кнопка только подсвечивается фиолетовым.

**Приоритет:** Желательный  
**Окружение:** Яндекс.Браузер 25.4.1.1024 (1280×720), Firefox 137.0.2 (1920×1081)

### BLK-73: Некорректный подсчёт ингредиентов в ленте заказов
**Краткое описание:**  
Каунтер "+N" для ингредиентов отображается на неправильном элементе.

**Шаги воспроизведения:**
1. Авторизоваться
2. Добавить в конструктор >7 ингредиентов
3. Оформить заказ
4. Перейти в "Ленту заказов"

**Ожидаемый результат:**  
Каунтер "+N" отображается на 6-м элементе при >5 ингредиентах.

**Фактический результат:**  
Каунтер отображается на 5-м элементе.

**Приоритет:** Желательный  
**Окружение:** Яндекс.Браузер 25.4.1.1024 (1280×720), Firefox 137.0.2 (1920×1081)
