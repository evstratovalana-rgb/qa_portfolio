# Примеры пунктов чек-листа для форм заказа самоката

**Объект тестирования:** Веб-формы «Для кого самокат» и «Про аренду» сервиса аренды.

**Примеры проверок:**

### 1. Форма «Для кого самокат»

#### UI и верстка:
- Логотип «Яндекс.Маршруты» отображается в левой части хэдера
- Кнопка «Заказать» в хэдере имеет белый текст на черном фоне с закругленными углами
- Все поля формы имеют одинаковую ширину и вертикальные отступы
- Незаполненные обязательные поля отображают серую звездочку и плейсхолдер

#### Валидация полей:
- **Поле «Имя»:** При вводе `Ольга1` появляется ошибка «Введите корректное имя»
- **Поле «Телефон»:** Валидный номер `89991234567` проходит проверку; `abc` вызывает ошибку
- **Поле «Станция метро»:** Ввод «Арб» показывает подсказку «Арбатская»
- **Адрес:** `ул. Ленина, 15-а` принимается; `ул. Ленина!` вызывает ошибку

#### Логика:
- Кнопка «Дальше» неактивна при пустой форме
- Кнопка «Дальше» становится активной при заполнении всех обязательных полей
- Клик по активной кнопке «Дальше» ведет к форме «Про аренду»

### 2. Форма «Про аренду»

#### Функциональность:
- Кнопка «Назад» возвращает к форме «Для кого самокат» с сохраненными данными
- В календаре нельзя выбрать сегодняшнюю или прошедшую дату
- Открытый календарь показывает текущий месяц
- Выбранная дата автоматически подставляется в поле

#### Валидация:
- Без выбранного срока аренды и с заполненной датой → ошибка «Введите корректный срок аренды»
- Без выбранной даты и с заполненным сроком → ошибка «Введите корректную дату доставки»
- При заполнении всех обязательных полей → появляется окно подтверждения заказа

#### Комплексные сценарии:
- При незаполненных полях даты и срока → две ошибки одновременно
- При выбранных дате, сроке и цвете самоката → успешное оформление заказа

---

**Ключевые проверяемые аспекты:**
- Валидация полей ввода (граничные значения, спецсимволы, форматы)
- Состояния кнопок (активность, hover-эффекты)
- Работа календаря и выпадающих списков
- Навигация между шагами формы
- Отображение сообщений об ошибках
- Сохранение данных при возврате
