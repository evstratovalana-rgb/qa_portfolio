# Примеры баг-репортов для сервиса аренды самокатов

## Категория 1: Баги веб-формы заказа

### БР-1: Несоответствие текста плейсхолдера в поле адреса
**Приоритет:** Желательный  
**Статус:** Открыт  
**Окружение:** Chrome 141, Яндекс.Браузер 25, разрешения 1920x1080 и 1280x720

**Предусловие:** Открыта форма "Для кого самокат"

**Шаги воспроизведения:**
1. Открыть форму заказа
2. Найти поле "Адрес: куда привезти самокат"
3. Проверить текст плейсхолдера

**Ожидаемый результат:** Плейсхолдер "Адрес: куда привезти самокат"
**Фактический результат:** Плейсхолдер "Адрес: куда привезти заказ"

## Категория 2: Баги логики форм

### БР-8: Отсутствие сообщения об ошибке при незаполненном сроке аренды
**Приоритет:** Стандартный  
**Статус:** Открыт  
**Окружение:** Chrome 141, Яндекс.Браузер 25

**Предусловие:**
1. Заполнена форма "Для кого самокат" (Иван Петров, адрес, станция, телефон)
2. Открыта форма "Про аренду"

**Шаги воспроизведения:**
1. Выбрать завтрашнюю дату
2. Выбрать цвет "серая безысходность"
3. Не выбирать срок аренды
4. Нажать "Заказать"

**Ожидаемый результат:** Отображается ошибка "Введите корректный срок аренды"
**Фактический результат:** Ошибка не отображается

## Категория 3: Баги мобильного приложения курьера

### БР-18: Задвоение карточек заказа
**Приоритет:** Стандартный  
**Статус:** Открыт  
**Окружение:** POCO X6 Pro 5G, Android 15, приложение Яндекс.Самокат 2.0

**Предусловие:**
1. Курьер авторизован в приложении
2. Клиент создал новый заказ
3. Открыт экран "Все"

**Шаги воспроизведения:**
1. Найти заказ из предусловия
2. Нажать "Принять"
3. Подтвердить "Да"
4. Перейти в список "Мои"

**Ожидаемый результат:** Одна карточка заказа в разделе "Мои"
**Фактический результат:** Две карточки одного заказа

### БР-23: Обрезание длинных текстов в карточке заказа
**Приоритет:** Блокирующий  
**Статус:** Открыт  
**Окружение:** POCO X6 Pro 5G, Android 15, приложение Яндекс.Самокат 2.0

**Предусловие:**
1. Создан заказ с длинными значениями:
   - Адрес: "Центральный проезд Хорошёвского Серебряного Бора 2"
   - Станция: "Бульвар Адмирала Ушакова"
   - Комментарий: "Привет, друг!"

**Шаги воспроизведения:**
1. Найти заказ в списке
2. Открыть полную карточку

**Ожидаемый результат:** Все данные отображаются полностью
**Фактический результат:** 
- Станция: "Бульвар Адмирала Ушак" (обрезано)
- Адрес: "Центральный проезд Хорошёвс" (обрезано)
- Комментарий: отсутствует полностью
