## **Критические пользовательские сценарии**

### 1. **Безопасное пожертвование:**
- Зайти на сайт → проверить HTTPS (SEC-01)
- Найти карточку в "Нужна помощь" → клик "ПОМОЧЬ" (CONT-03)
- Переход на страницу семьи → проверка HTTPS
- Клик "ПОМОЧЬ С КАРТЫ" → переход на защищённую оплату (FORM-01)

### 2. **Мобильная навигация:**
- Открыть сайт на телефоне → открыть боковое меню (SIDE-01)
- Выбрать "Контакты" → переход по HTTPS (SIDE-03)
- Кликнуть на телефон → открытие приложения звонка (FOOT-03)

### 3. **Работа с медиа:**
- Открыть раздел "Фотоотчёт" → свайп влево (GALL-01)
- Клик на фото → открытие модального окна (GALL-02)
- Закрытие окна → возврат в галерею

## **Особенности тестирования**

### Платформы:
- **Web**: Проверка скролла мышью, drag-n drop, hover-эффекты
- **Web-mobile**: Свайпы, тапы, адаптация под ориентацию
- **All**: Кросс-платформенные проверки (HTTPS, Cookie)

### Приоритеты:
- **HIGH**: Безопасность, основные переходы, критичная функциональность
- **MEDIUM**: UI/UX, адаптивность, работа медиа
- **LOW**: Визуальные детали, соответствие макетам
